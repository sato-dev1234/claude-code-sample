# Anti-Patterns

| Section | ❌ Avoid | ✅ Do Instead |
|---------|---------|---------------|
| 概要 | "○○関数を修正" | "貸出履歴で会員種別が解決されない不具合を修正" |
| 問題点 | "実装にバグがありました" | AC基準の表形式で記述 |
| 原因 | "`file:line`で○○していなかった" | AC Gap Tableで期待と実際のギャップを記述 |
| 解決策 | "フィルタ条件を修正し..." | Before/After Tableで変更内容を記述 |
| 確認項目 | "動作確認" | 具体的なチェック項目を列挙 |
| 確認手順 | "画面で操作する" | 番号付きステップで再現可能に |
| リグレッション | "全機能が動く" | 影響を受ける可能性のある具体的機能を列挙 |
